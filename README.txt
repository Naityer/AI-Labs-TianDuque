# AI Labs – Tian Duque

Este repositorio reúne diversos proyectos realizados durante mis estudios de Ingeniería de Computadores, enfocados en Inteligencia Artificial, Aprendizaje por Refuerzo, Regresión, Algoritmos de Juegos y Lógica Computacional.

## Proyectos incluidos

| Proyecto                        | Descripción breve |
|-------------------------------|-------------------|
| **GreedyMaze**                | Algoritmo voraz para resolver laberintos. |
| **EARIN Regression**          | Regresión en 2D con algoritmos genéticos. |
| **Rastrigin RS**              | Optimización con búsqueda aleatoria. |
| **MiniMax Connect4**          | Implementación de IA para el juego Connect 4. |
| **Prolog Interval**           | Generación y validación de gramáticas macro. |
| **Q-Learning MountainCar**    | Algoritmo de Q-Learning usando Gymnasium. |
| **MLP MNIST**                 | Clasificación de dígitos con red neuronal multicapa. |

## Tecnologías utilizadas

- Python
- Prolog
- Gymnasium (OpenAI)
- Scikit-learn
- NumPy, Matplotlib, etc.

## Estructura

Cada carpeta contiene su propio `README.md`, código fuente y scripts para entrenamiento o pruebas.
